## 📌 About the Project
This project analyzes the **Global Superstore dataset** to uncover hidden patterns in:
- Sales performance 💰  
- Shipping efficiency 🚚  
- Customer segments 👥  
- Regional profitability 🌍  

👉 The goal is to transform raw data into **actionable business insights**.

---

## 🧠 Key Insights
✔️ Standard Class has the longest delivery time  
✔️ Technology category dominates sales  
✔️ Discounts reduce profit significantly  
✔️ Strong relationship between Sales & Shipping Cost  
✔️ Profit varies significantly across regions
